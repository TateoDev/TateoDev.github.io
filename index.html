Pest Control

Hide and seek game made in collaboration with development partner. I did the scripting while he provided assets and maps.

  • The hiders are small while the seekers are regular size.
  • Hiders are given a short countdown before the seeker begins.
  • Seekers are able to catch hiders using their fly swatter
  • Players can buy new cosmetic swatters using tokens earned during the round
  • Swatting is done using using magnitude checks and raycasts, ensuring all cosmetics are fair
  • All data is stored using ProfileStore and delivered to client using Replica

Technologies used: ProfileStore/Replica, Team Create
